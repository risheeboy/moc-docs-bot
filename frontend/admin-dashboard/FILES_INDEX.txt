========== KEY FEATURES ==========

1. Dashboard:       Real-time KPIs (query volume, response time, sessions, satisfaction)
2. Analytics:       Language dist, topic dist, model usage, satisfaction trends
3. Documents:       Upload, manage, delete documents with status tracking
4. Scrape Jobs:     Trigger and monitor web scraping for 30 Ministry sites
5. Conversations:   Browse and search user conversation sessions
6. Feedback:        Sentiment analysis with pie and bar charts
7. System Config:   Manage RAG, LLM, session settings and feature flags
8. Model Monitor:   Langfuse integration for per-model metrics
9. Audit Log:       Searchable admin activity tracking
