# Required Deliverables

**Source:** Requirements.pdf pages 12-13

## Software Deliverables

1. **Fully functional AI-powered search interface** (Unified Semantic Search page)
2. **Fully functional conversational chatbot** (embeddable widget)
3. **Admin dashboard** for system management and analytics
4. **All backend services** (API Gateway, RAG, LLM, Speech, Translation, OCR, Data Ingestion)
5. **Source code** — complete source code with inline documentation
6. **Docker Compose deployment** — one-command deployment of entire system

## Documentation Deliverables

All documentation must be provided in both **English and Hindi** where applicable.

| # | Deliverable | Description |
|---|---|---|
| 1 | **Source Code Documentation** | Inline docstrings + architecture documentation |
| 2 | **API Documentation** | OpenAPI/Swagger specification + human-readable API reference with request/response examples |
| 3 | **Deployment Guide** | Step-by-step deployment instructions for NIC/MeitY Data Centre, including prerequisites (hardware, OS, GPU drivers, Docker, NVIDIA toolkit) |
| 4 | **User Manual (English)** | End-user guide for using the chatbot and search interfaces |
| 5 | **User Manual (Hindi)** | Hindi version of the user manual |
| 6 | **Admin Guide** | Guide for admin dashboard usage, document management, system configuration |
| 7 | **Training Material** | Training curriculum and materials for Ministry staff (admin + end-user) |
| 8 | **Backup & Recovery Procedures** | Documented backup schedules and disaster recovery plan |
| 9 | **Monitoring Guide** | How to use Prometheus/Grafana dashboards, set up alerts, review Langfuse |

## Test Reports (Mandatory Deliverables)

| # | Report | Content |
|---|---|---|
| 1 | **UAT Report** | User Acceptance Testing results, WCAG 2.1 AA compliance, GIGW compliance |
| 2 | **Performance Test Report** | Load testing results showing p50/p95/p99 latency, throughput, concurrent user handling |
| 3 | **Security Audit Report** | OWASP API Top 10 assessment, vulnerability scanning results, penetration testing summary, dependency CVE audit |
| 4 | **Integration Test Report** | End-to-end test results across all service flows |

## Compliance Reports

| # | Report | Content |
|---|---|---|
| 1 | **GIGW Compliance Report** | Checklist of GIGW requirements and compliance status |
| 2 | **WCAG Compliance Report** | WCAG 2.1 AA accessibility audit results |
| 3 | **Data Sovereignty Report** | Documentation of data residency and sovereignty controls |

## Intellectual Property
