# Completion Schedule & Milestones

**Source:** Requirements.pdf page 16

## Overall Timeline

The RFP specifies a phased delivery approach. Key milestones:

| Phase | Milestone | Timeline |
|---|---|---|
| Phase 1 | **Proof of Concept (PoC)** | Within 8 weeks of contract signing |
| Phase 2 | **Full Development & Deployment** | Within 16 weeks of contract signing |
| Phase 3 | **UAT & Go-Live** | Within 20 weeks of contract signing |
| Phase 4 | **AMC Period** | 3 years from Go-Live |

## Phase 1: Proof of Concept (8 weeks)

Deliver working prototype demonstrating:
- Chatbot with basic Q&A capability on culture.gov.in content
- Search interface with semantic search across at least 6 PoC websites:
  1. indianculture.gov.in
  2. mgmd.gov.in
  3. vedicheritage.gov.in
  4. museumsofindia.gov.in
  5. gyanbharatam.com
  6. abhilekh-patal.in
- Hindi + English voice interaction
- Basic admin dashboard

## Phase 2: Full Development (16 weeks)

- Scale to all 30 ring-fenced websites
- Full feature set: OCR, translation (22 languages), sentiment analysis
- Fine-tuned models on domain data
- Complete RBAC implementation
- Performance optimization (< 5s p95)
- Full monitoring and alerting setup

## Phase 3: UAT & Go-Live (20 weeks)

- User Acceptance Testing with Ministry stakeholders
- Security audit and penetration testing
- GIGW compliance verification
- WCAG 2.1 AA accessibility audit
- Performance/load testing under realistic conditions
- All documentation deliverables completed
- Production deployment on NIC/MeitY infrastructure

## Phase 4: AMC (3 years)

- Bug fixes and maintenance
- Security patches
- Model retraining and optimization
- Content re-indexing as websites update
- Performance monitoring and tuning
- Support for adding new data sources
